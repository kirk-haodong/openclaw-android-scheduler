## [1.0.0] - 2026-03-18

First release of OpenClaw Android Scheduler.

### Added
- Initial release with Python daemon for cron-like scheduling on Android/Termux/PRoot
- Timezone handling (Beijing Time UTC+8)
- State tracking to prevent duplicate executions
- Configurable task scheduling via Python dictionary
- Comprehensive documentation (README, SKILL, Changelog)
